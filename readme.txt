== Description ==
Clean, Cutting-Edge theme conceived for quick loading, minimal coding and great user experience. Easy to visualise in any devise, not only because the mobile-first approach implemented for its creation but also the clear intention of getting a theme for visual impact to show off in a fast-loading canvas. Enjoy!

* Responsive Layout
* Custom Colors
* Social Links
* Menu Description
* Post Formats

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's ZIP file. Click Install Now.
3. Click Activate to use your new theme right away.
4. We suggest to install: Simple Share Buttons Adder, Owl Carousel Plugins & (as an option) Responsive Lightbox. Enjoy!

== Frequently Asked Questions ==

= How do I add the Social Links to the sidebar? =

1. Create a new Custom Menu, and assign it to the Social Links Menu location.
2. Add links to each of your social services using the Links panel.
3. Icons for your social links will automatically appear if it's available.

Available icons: (Linking to any of the following sites will automatically display its icon in your social menu).

* Facebook
* Google+
* Instagram
* LinkedIn
* WordPress
* YouTube
